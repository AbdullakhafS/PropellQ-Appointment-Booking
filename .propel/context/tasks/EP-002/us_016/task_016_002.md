# Task 002: Implement Verification Update API and Audit Logging

**Task ID:** task_016_002  
**User Story:** US-016 (Flag Unverified Insurance for Staff Review)  
**Epic:** EP-002  
**Priority:** CRITICAL  
**Story Points:** 2  
**Owner Role:** Backend Developer  
**Status:** Not Started

## Objective

Implement manual verification update APIs and audit trail logging so staff actions are tracked and reviewable.

## Requirements

1. Endpoint:
   - `PATCH /api/insurance/{insuranceVerificationId}/verify`
   - Payload: `status`, `verification_method`, `notes`

2. Update behavior:
   - Update verification status (`verified` or `manual_review`)
   - Set `verified_by_staff_id` from authenticated user
   - Set `verified_at` to current timestamp

3. Audit logging:
   - Create `InsuranceVerificationAudit` table entry on every status change
   - Store previous_status, new_status, verified_by_staff_id, method, notes, timestamp

4. Batch support:
   - Add `PATCH /api/insurance/verify/batch` for multiple IDs
   - Return success/failure summary per record

5. Authorization and validation:
   - Billing/admin role required
   - Reject invalid status transitions and missing method

## Implementation Steps

1. Add audit table migration and indexes.
2. Implement single-record verify endpoint.
3. Implement batch verify endpoint.
4. Ensure transactional integrity for update + audit insert.
5. Add integration tests for both endpoints.

## Acceptance Criteria

- [ ] Single verify endpoint updates status and metadata
- [ ] Batch verify endpoint updates multiple rows
- [ ] Audit record created for each status change
- [ ] Unauthorized users cannot verify records
- [ ] Invalid requests return clear error responses

## Dependencies

- InsuranceVerifications data model from US-015
- Role-based access control

## Validation

- [ ] Verify endpoint updates status correctly
- [ ] Batch update logs all audit entries
- [ ] Audit history reflects accurate change timeline

## Definition of Done

- Manual verification APIs and audit logging are implemented and tested.
