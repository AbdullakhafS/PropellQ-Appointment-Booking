# Task 005: Add Batch Verification Actions and CSV Export

**Task ID:** task_016_005  
**User Story:** US-016 (Flag Unverified Insurance for Staff Review)  
**Epic:** EP-002  
**Priority:** MEDIUM  
**Story Points:** 1  
**Owner Role:** Full-Stack Developer  
**Status:** Done

## Objective

Enable staff efficiency features: batch "mark as verified" actions and CSV export for offline processing.

## Requirements

1. Batch actions in UI:
   - Multi-select checkboxes per row
   - "Mark as Verified" bulk button
   - Confirmation dialog before applying bulk updates

2. Batch API integration:
   - Use `PATCH /api/insurance/verify/batch`
   - Show per-record success/failure summary after completion

3. CSV export:
   - Export currently filtered result set
   - Include columns: patient name, insurance name, member ID, confidence score, status, appointment date
   - File naming with date/time stamp

4. Safeguards:
   - Disable batch action when no rows selected
   - Limit large batch size or process in chunks

## Implementation Steps

1. Add row selection and bulk action controls.
2. Integrate batch verify endpoint.
3. Add CSV export action with active filter context.
4. Add user feedback for success/failure.
5. Add tests for bulk and export actions.
