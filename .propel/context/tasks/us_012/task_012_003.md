# Task 003: Create Figma Mockups (Progress Indicator & UI)

**Task ID:** task_012_003  
**User Story:** US-012 (Design Multi-Step Chatbot Conversation Flow)  
**Epic:** EP-002  
**Priority:** HIGH  
**Story Points:** 2  
**Owner Role:** UX / Product Designer  
**Status:** Not Started

## Objective

Create Figma mockups for chatbot UI elements including conversation display, progress indicators, and visual feedback for different conversation stages.

## Requirements

1. **Chatbot Message Display**
   - Message bubble styles: assistant (left-aligned, light background), user (right-aligned, blue)
   - Typography: clear, readable font sizes for desktop and mobile
   - Message timestamps and loading states

2. **Progress Indicator**
   - Stage-based progress: "Step 2 of 7" text indicator
   - Visual progress bar showing completion %
   - Estimated remaining time: "About 4 more minutes"
   - Stage labels visible (Greeting → Chief Complaint → History → etc.)

3. **Input States**
   - Empty state: chatbot greeting with input field ready
   - Loading state: typing indicator (animated dots) while waiting for response
   - Error state: retry button, fallback to manual form option
   - Completion state: summary with edit buttons and confirm/submit

4. **Mobile Responsiveness**
   - Desktop (1024px): full-width chatbot panel
   - Tablet (768px): slightly narrower panel with optimized spacing
   - Mobile (375px): full-screen chatbot experience
   - Touch-friendly button sizes (48px minimum)

5. **Visual Feedback**
   - Emphasis on current question
   - Previous answers visible above (conversation history)
   - Clear distinction between user input and chatbot responses
   - Visual cues for optional vs. required fields

6. **Accessibility**
   - High contrast between text and background
   - ARIA-compatible labels and descriptions
   - Keyboard navigation indicators
   - Screen reader friendly markup

## Implementation Steps

1. Set up Figma file with design system tokens (colors, typography, spacing)
2. Create base chatbot component: message bubble, input field, buttons
3. Design progress indicator component (text, progress bar, stage labels)
4. Create mockups for key conversation stages: greeting, question, response, summary
5. Design loading, error, and completion states
6. Implement mobile/tablet/desktop responsive layouts
7. Add interactive prototypes showing flow transitions
8. Document component specs and interaction patterns
9. Get design review feedback
10. Finalize Figma file for handoff to development

## Acceptance Criteria

- [ ] Chatbot message display mockups created for desktop, tablet, mobile
