# Task 003: Add Audited Deletion Controls for Expired Records

Task ID: task_076_003
User Story: US-076
Epic: EP-007
Priority: HIGH
Story Points: 1
Owner Role: Security / Compliance Engineer
Status: Done

## Objective
Restrict deletion to expired records and record approval/audit evidence for removals.

## Acceptance Criteria
- [x] Only expired records can be deleted.
- [x] Deletions are logged and approval-controlled.

## Definition of Done
Deletion controls are compliant and traceable.

## Implementation
- `app/src/audit_storage.py` — `DeletionApproval` dataclass, `AuditedDeletionController.request_deletion()` enforces archive-tier membership + 7-year expiry + non-empty approval, emits `LIFECYCLE_DELETE_APPROVED` / `LIFECYCLE_DELETE_DENIED` audit events.
