# Task 003: Create and Dispatch Waitlist Offers Automatically

Task ID: task_042_003  
User Story: US-042 (Implement Waitlist Processing on Slot Release)  
Epic: EP-004  
Priority: HIGH  
Story Points: 1  
Owner Role: Backend Developer  
Status: Done

## Objective

Automatically create offers for selected waitlist entries and dispatch notifications through the existing notification service.

## Requirements

1. Offer creation:
   - Generate offer record linked to slot and waitlist entry
   - Set offer state to pending
   - Set expiry timestamp
2. Notification dispatch:
   - Send offer notification to selected patient
   - Include action path for accept/decline
3. Tracking:
   - Store delivery status and offer metadata

## Implementation Steps

1. Implement offer creation service call.
2. Integrate with notification provider/service.
3. Persist dispatch status for observability.
4. Add integration tests for offer + notification flow.

## Acceptance Criteria

- [x] Offer is created for selected candidate.
- [x] Waitlist entry state updates to pending offer.
- [x] Patient notification is dispatched.
- [x] Offer metadata and dispatch status are persisted.

## Dependencies

- Task 042_002 candidate selection.
- Notification service integration.

## Validation

- [x] Offer record exists with correct slot linkage.
- [x] Notification event payload is correct.

## Definition of Done

- Automated offer creation and dispatch are implemented.
