# Task 001: Implement Drag-and-Drop Queue Reordering UI

**Task ID:** task_035_001  
**User Story:** US-035 (Implement Drag-Drop Queue Reordering)  
**Epic:** EP-004  
**Priority:** CRITICAL  
**Story Points:** 2  
**Owner Role:** Frontend Developer  
**Status:** Done

## Objective

Enable staff to reorder same-day queue items through an intuitive drag-and-drop interaction.

## Requirements

1. Add drag-and-drop interaction to queue list.
2. Use UI-stack-consistent DnD library.
3. Provide clear visual cues during drag (placeholder/highlight).
4. Support keyboard-accessible fallback controls where feasible.
5. Maintain stable item identity to avoid reorder glitches.

## Implementation Steps

1. Integrate drag-drop library in queue component.
2. Implement reorder handlers for drag start/drop.
3. Update local state optimistically on drop.
4. Add accessibility labels and focus behavior.
5. Add component tests for reorder interaction.

## Acceptance Criteria

- [x] Queue items can be reordered via drag-and-drop
- [x] Visual drag/drop feedback is clear
- [x] Reordered list updates immediately in UI
- [x] Basic accessibility behavior is present

## Dependencies

- Queue UI from EP-004
- Real-time queue base from US-034

## Validation

- [x] Drag-drop works with multiple queue lengths
- [x] No item duplication/loss during reorder

## Definition of Done

- Drag-and-drop reordering works reliably in the staff queue UI.
