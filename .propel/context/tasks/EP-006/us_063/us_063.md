# US-063: Display Appointment Utilization Analytics

**Story ID:** US-063  
**Epic:** EP-006 (Patient Portal & Analytics)  
**Priority:** MEDIUM  
**Story Points:** 3  
**Owner Role:** Product / Data Analyst  
**Status:** Draft

## User Story

As an **admin operations manager**, I want appointment utilization analytics so that I can see how provider time and room capacity are being used.

## Business Value

- Helps identify underutilized providers or resources
- Supports schedule optimization and resource allocation
- Reduces wasted operational capacity

## Scope

In scope:
- Utilization metrics for provider or location slots
- Charts for booked vs available appointments
- Filters by provider, specialty, and location

Out of scope:
- Detailed staff time tracking beyond appointments
- Real-time room occupancy integration

## Acceptance Criteria

1. Given an admin views the dashboard, then utilization metrics are visible.
2. Given filters are applied, then utilization data updates accordingly.
3. Given the admin views the charts, then booked vs available slot comparisons are shown.
4. Given a provider or location is selected, then utilization is scoped appropriately.

## Technical Notes

- Calculate utilization as booked slots divided by total available slots.
- Provide clarity for partial-day and provider-specific views.
- Use aggregated data to avoid frequent heavy queries.

## Dependencies

- Appointment availability and booking data
- Analytics data store or service

## Definition of Done

- Utilization analytics display on admin dashboard
- Filters work correctly for provider and location scopes
- Charts and metrics render as expected
- Acceptance criteria validated with sample data
