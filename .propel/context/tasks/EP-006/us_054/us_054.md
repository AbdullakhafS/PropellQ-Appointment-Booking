# US-054: Display Upcoming Appointments with Actions

**Story ID:** US-054  
**Epic:** EP-006 (Patient Portal & Analytics)  
**Priority:** HIGH  
**Story Points:** 3  
**Owner Role:** Product / Platform Engineer  
**Status:** Draft

## User Story

As a **patient**, I want to see my upcoming appointments with action buttons so that I can reschedule, cancel, or view details quickly.

## Business Value

- Enables faster appointment management
- Reduces no-shows by making actions obvious
- Gives patients confidence in their upcoming care

## Scope

In scope:
- Upcoming appointment list with provider, date, time, and location
- Reschedule and cancel actions for eligible appointments
- Status badges for confirmed, pending, or check-in ready appointments

Out of scope:
- Complex rescheduling workflows
- In-depth provider profiles

## Acceptance Criteria

1. Given the patient has upcoming appointments, then each appointment is displayed clearly with key details.
2. Given an appointment is eligible for changes, then reschedule and cancel buttons are available.
3. Given the appointment has already passed, then it is not displayed in the upcoming list.
4. Given the patient clicks an action, then the appropriate workflow is initiated.

## Technical Notes

- Pull upcoming appointment data from the booking API.
- Display status and location in compact patient-friendly format.
- Disable actions when not allowed by policy.

## Dependencies

- Appointment booking backend from EP-001
- Reschedule/cancel service authorization from EP-005

## Definition of Done

- Upcoming appointment section implemented on the dashboard
- Action buttons appear when permitted
- Appointment details load correctly and update on changes
- Acceptance criteria verified with sample patient data
