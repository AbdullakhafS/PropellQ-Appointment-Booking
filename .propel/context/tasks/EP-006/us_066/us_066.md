# US-066: Display AI-Human Agreement Rate

**Story ID:** US-066  
**Epic:** EP-006 (Patient Portal & Analytics)  
**Priority:** MEDIUM  
**Story Points:** 3  
**Owner Role:** Product / Data Analyst  
**Status:** Draft

## User Story

As an **admin operations manager**, I want to see AI-human agreement rate metrics so that I can monitor the accuracy of clinical coding and data extraction.

## Business Value

- Helps validate AI-assisted clinical intelligence
- Supports trust in automated coding workflows
- Enables targeted improvement of AI models and review processes

## Scope

In scope:
- Agreement rate KPI card comparing AI suggestions to human review
- Trend chart over selectable date range
- Breakdown by document type or clinical workflow

Out of scope:
- Model retraining or AI diagnostics tools
- Real-time prediction accuracy dashboards

## Acceptance Criteria

1. Given the admin views the dashboard, then the AI-human agreement card is visible.
2. Given the trend chart loads, then it shows agreement percentage over time.
3. Given filters are applied, then the agreement metrics update accordingly.
4. Given a category breakdown is selected, then the dashboard updates the display.

## Technical Notes

- Compute agreement rate from reviewed AI suggestions and confirmed human outcomes.
- Use aggregated analytics data for dashboard performance.
- Clearly explain the metric definition in the dashboard UI.

## Dependencies

- Clinical AI review and human validation data from EP-003
- Analytics aggregation service

## Definition of Done

- AI-human agreement rate metrics display on admin dashboard
- Trend chart and filter behavior work correctly
- Acceptance criteria validated with sample review data
