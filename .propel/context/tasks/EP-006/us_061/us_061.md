# US-061: Display No-Show Rate & Trends

**Story ID:** US-061  
**Epic:** EP-006 (Patient Portal & Analytics)  
**Priority:** MEDIUM  
**Story Points:** 3  
**Owner Role:** Product / Data Analyst  
**Status:** Draft

## User Story

As an **admin operations manager**, I want to see no-show rates and trend charts so that I can identify patterns and reduce appointment no-shows.

## Business Value

- Helps optimize clinic scheduling and patient outreach
- Reveals problematic time slots or provider patterns
- Supports efforts to reduce lost revenue from missed visits

## Scope

In scope:
- No-show rate card with current rate and change indicator
- Trend chart over selectable date range
- Comparison to previous period or baseline

Out of scope:
- Predictive no-show modeling
- Root-cause analysis beyond KPI reporting

## Acceptance Criteria

1. Given the admin views the dashboard, then the no-show rate card is visible.
2. Given the trend chart loads, then it shows no-show percentage over time.
3. Given a date range filter is applied, then the trends update accordingly.
4. Given the dashboard compares to a prior period, then the comparison is displayed clearly.

## Technical Notes

- Calculate no-show rate as missed appointments divided by scheduled appointments.
- Use cached or aggregated data for performance.
- Provide clear labeling for trends and changes.

## Dependencies

- Appointment status data from EP-001/EP-004
- Analytics data store or aggregation service

## Definition of Done

- No-show metrics display on admin dashboard
- Trend chart updates with filters
- Comparison values shown correctly
- Acceptance criteria validated with sample analytics data
