# US-064: Display Intake Completion Rates

**Story ID:** US-064  
**Epic:** EP-006 (Patient Portal & Analytics)  
**Priority:** MEDIUM  
**Story Points:** 3  
**Owner Role:** Product / Data Analyst  
**Status:** Draft

## User Story

As an **admin operations manager**, I want intake completion metrics so that I can monitor how many patients complete pre-visit intake and identify process gaps.

## Business Value

- Increases intake completion rates and visit readiness
- Helps track staff follow-up needs for incomplete intake
- Supports patient experience optimization

## Scope

In scope:
- Intake completion rate KPI card
- Trend and breakdown by appointment type or clinic
- Filter by date range and provider

Out of scope:
- Deep intake form analytics per field
- Real-time intake nudges or reminders

## Acceptance Criteria

1. Given the admin views the dashboard, then intake completion rate metrics are visible.
2. Given a date range filter is applied, then intake completion data updates accordingly.
3. Given intake completion is low, then the dashboard highlights the metric for review.
4. Given filters are changed, then the KPI reflects the selected subset.

## Technical Notes

- Calculate intake completion as completed intake forms divided by scheduled visits.
- Use appointment and intake status data from EP-002.
- Display both current rate and trend over time.

## Dependencies

- Patient intake completion data from EP-002
- Appointment scheduling data from EP-001
- Analytics data service

## Definition of Done

- Intake completion metrics display on admin dashboard
- Filters update the displayed data correctly
- Acceptance criteria validated with representative metrics
